# Alibaba LabelX ASR 转写资料

## 当前状态

- 本目录用于沉淀 LabelX ASR 转写的平台资料与契约说明。
- 当前代码仍在：
  - `extension/sites/alibaba-labelx/asr-transcription/`
- 当前页面结构样本和网络契约资料仍不完整，需后续在真实页面补充。

## 当前业务口径（与扩展运行时一致）

- 当前阶段只做基础转写。
- 一条音频对应一个完整文本框。
- 不实现时间戳、说话人区分、AI 初稿、AI 校对、AI 格式化。
- 保存方式以平台自动保存为准。
- 不新增自定义后端保存接口。
- 不构建/注入自定义保存 payload。
- 不自动提交、不自动领取、不自动流转。
- 不保留全页批量修改动作（全页标有效并填充 / 全页去空格 / 全页校验自动修复）。

## 当前代码状态（2026-05-08）

- 已从扩展侧删除旧自动化与旧保存链路入口：
  - 自动提交、智能提交、批量提交
  - 自动领取、自动抢单、自动流转、自动跳转下一任务
  - 保存后 reload、自定义保存 payload 注入、强制保存
  - AI 标点、排行榜、导出统计
- 已物理删除 `asr-transcription` 下对应旧文件，不再保留空壳兼容。
- 若后续要恢复任一旧能力，必须走“新需求 -> 新设计 -> 新实现 -> 新验收”，不直接恢复旧脚本。
