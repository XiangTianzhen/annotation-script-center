# 台州话原始听音规则

- 每段音频仅调用一次 Qwen Omni。
- 扩展/API 将模型输出逐字映射为 `listenText` 兼容字段；该字段不是模型输出格式。
- 默认 Prompt 要求原样记录听到的台州话，不翻译、不润色、不补写、不删减；默认输出为当前片段最终转写文本，纯静音或完全听不清时输出空字符串。
- 非空自定义 Prompt 会原样作为完整 systemPrompt，完全决定模型语义和输出格式。后端只附带片段、时间范围、字段/编辑上下文和规则资料已加载标识，不追加不翻译、原样记录、纯文本或其他规则。
- 后端不会改数字、规整标点、压缩重复字、trim 或折叠模型输出空白。
- 不输出唱歌、方言类别、风险、复核、强制填入或自动填入判断。
