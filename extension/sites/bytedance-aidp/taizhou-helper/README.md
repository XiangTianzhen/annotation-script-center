# 台州话 AIDP 运行时

## 定位

台州话脚本用于诊断百炼模型与标注平台模型的差异：每段音频只请求一次 Qwen Omni，后端再将字符串原始输出逐字符映射为 `listenText` 供平台文本框使用。

## 原始听音直填契约

- 后端成功结果的唯一业务文本字段是 `listenText`；它是扩展/API 兼容字段，不是模型要求返回的 JSON 字段。
- `listenText` 按原样传递；不做 trim、空白折叠、普通话转换、数字改写、标点规整或重复压缩。
- 后端只在原始模型输出为字符串时逐字符映射 `listenText`；空字符串或非字符串不写入 textarea 或暂存数据。所有非空字符串均会原样写入，包括模型意外输出的 JSON、Markdown 或解释文字。
- 单段与行内识别通过真实 textarea 的输入事件直接写入 `listenText`。
- 批量只处理当前题当前页选中的段，收集非空 `listenText` 后一次通过 `SubmitTempItemAnswer` 写入 `regions[*].txt`。
- 批量停止后不再发新请求；已在途结果不会产生二次写回操作。
- 全程不触发保存、提交或切题，且不修改 `regions[*].ms`。

## 界面与设置

- 行内按钮始终为“识别音频”；不保留待填入、强制填入或风险提示状态。
- AI 信息区显示“原始听音”、调用消耗以及 `raw/debug`。
- Options“基础设置”的“录音平台数据库内部 taskId”默认空且保存时去除首尾空白；空值不显示导入按钮，也不查询结果。
- 配置 taskId 后，详情页顶部只在 Search Item 与当前 Receive ItemID 一致且快照有效时启用“导入录音任务”。导入固定使用当前完整题目，不按画段拆分。
- 辅助面板“录音平台结果”只读显示 sourceItemId、itemCode、友好状态、完成文本和音频；每次进入匹配题目只自动查询一次，A→B→A 再次进入 A 时重新查询，连续停留不轮询。切题后的过期响应不渲染；结果音频只接受脚本中心预期相对 API 路径，不展示结果视频、不写回 AIDP。
- 台州详情页不显示“识别完成后自动填入”；保存的非空自定义 Prompt 不迁移、不覆盖，并原样作为完整 systemPrompt，完全决定模型语义和输出格式。后端只附带片段与标注上下文，不追加转换、听写或输出格式规则；清空后回退默认 Prompt；旧自动填入 storage 值不读取、不迁移。
- 其他 AIDP 脚本的设置不受影响。

## 本地验证

1. 在扩展管理页加载 `extension/`。
2. 刷新 AIDP 标注详情页。
3. 单段识别后，对比 textarea 与接口响应的 `listenText` 是否逐字符一致。
4. 批量只选择部分段，确认仅这些段的 `regions[*].txt` 暂存更新，且不提交、不切题。
5. 使用脱敏测试任务配置内部 taskId，确认按钮等待、导入成功、重复点击复用、结果单次自动查询与手动刷新；完成文本和音频不得出现在 AIDP 输入框或暂存请求中。
