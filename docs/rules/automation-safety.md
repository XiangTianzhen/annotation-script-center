# 自动化安全规则（项目级）

本文档定义跨平台通用自动化边界，不包含具体平台页面细节。

## 用户触发与默认边界

- AI 建议默认只作辅助，不自动执行页面状态流转。
- 仅在用户明确点击按钮或触发快捷键后，允许执行“填入输入框”类辅助动作。
- 未经当前 Prompt 明确授权，禁止自动保存、自动提交、自动领取、自动送审、自动审核、自动流转。

## 批量与连续处理约束

- 连续处理、批量填入、并发 AI 分析必须明确作用范围，默认只处理当前页，不跨页。
- 必须提供停止机制（按钮或快捷键）。
- 必须提供失败统计或失败提示，避免静默失败。
- 不得点击 checkbox，除非当前 Prompt 明确授权。
- 不得绕过平台原生禁用状态（disabled / readonly）。

## 安全与鉴权约束

- 不得硬编码 token、cookie、access_token、authorization。
- 前端不得持久化 API Key、cookie、token、完整签名 URL、完整音频/文件 URL。
- 日志必须脱敏，仅保留必要调试信息（如 requestId、状态码、耗时、模型名）。
